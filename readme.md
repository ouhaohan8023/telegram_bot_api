## 本项目实现以下功能

1. 尬聊(对接图灵api)。 触发：任意未定义信息
2. 汇率计算，主要计算汇率差消耗。触发：&金额&高汇率&低汇率
3. 模仿框架，写了一个简单的路由处理

## 操作
1. 绑定Webhook
2. 如果日志没有记录每次通讯的日志，那么需要删除Webhook之后，才可以调用getUpdates查看chat_id

## 项目构成
0. Composer
1. DotEnv
2. Monolog
